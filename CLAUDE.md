# FruitMap (열매맵) v2.0

## 프로젝트 개요
개신교 복음방 인도자가 "열매 체크리스트"(21개 항목 + MBTI)를 입력하면,
3-레이어 분석(6차원 + 교차패턴 + MBTI)으로 맞춤 상담 가이드 + 재배치된 커리큘럼 로드맵을 생성하는 웹앱.

## 기술 스택
- Next.js 14 (App Router), TypeScript, Tailwind CSS, Recharts, Vercel

## 핵심 구조
- 3-레이어 분석: Layer 1(WHAT) 6차원 + Layer 2(DEEP) 교차패턴 + Layer 3(HOW) MBTI
- 3축 상담 가이드: 유형별 원칙(7) × 고민별 콘텐츠(16) × 교차패턴 보완(9)
- 7개 매칭 유형별 커리큘럼 순서 재배치
- 탈락 키워드 경고 시스템 (🔴/🟠/⚠️ 3단계)
- v2.0 신규 필드: 수강동기, 외부정보접촉, 기간확답
- retention = 60%침요소 + 20%동기 + 20%확답 가중치

## 설계 원칙
- 모바일 퍼스트 (인도자 현장 사용)
- 클라이언트 사이드 계산 (DB 불필요, MVP)
- 데이터 분리 (커리큘럼, 상담DB, 매핑테이블 별도 파일)
- MBTI 선택 입력 (없어도 핵심 기능 작동)
- 한국어 전용, 탈락 데이터 기반 경고 (실증적)
