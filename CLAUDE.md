# FruitMap (열매맵)

## 프로젝트 개요
개신교 복음방 인도자가 "열매 체크리스트"(18개 항목 + MBTI)를 입력하면,
6차원 영적 성장 프로파일을 분석하고 **개인 맞춤 순서로 재배치된** 20과 커리큘럼 로드맵 + 교수법 가이드를 생성하는 웹앱.

## 기술 스택
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Recharts (레이더 차트)
- Vercel 배포

## 핵심 구조
- 2-레이어 분석: Layer 1(WHAT) 6차원 + Layer 2(HOW) MBTI 교수법
- 7개 매칭 유형별 커리큘럼 순서 재배치
- 오버라이드 규칙: 이단경험→18과 이동, 정신건강→13과 이동, 가족반대→19과 이동
- 압축형(⚡)은 1차/2차 구분

## 설계 원칙
- 모바일 퍼스트 (인도자가 현장에서 사용)
- 클라이언트 사이드 계산 (DB 불필요, MVP)
- 커리큘럼 데이터 분리 (향후 추가/수정 용이)
- MBTI는 선택 입력 (없어도 핵심 기능 작동)
- 한국어 전용
